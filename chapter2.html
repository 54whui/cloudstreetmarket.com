Chapter 2, Designing a Microservice Architecture with Spring MVC

Spring MVC Cookbook, Chapter 2 Designing a Microservice Architecture with Spring MVC, introduces the core principles of Spring MVC. We visit the framework request flow, the central role of the DispatcherServlet, and we demonstrate how to configure Spring MVC controllers and controller handler methods using an extended source of information related to controllers' annotations.

In the path of a Microservice architecture, we install Spring and Spring MVC across modules and web projects to build functional units, easy to deploy and easy to scale. We shape our application with a web module responsible for serving a Twitter Bootstrap template along with another web module dedicated to REST Web Services.

In this chapter, we finally learn how to transfer the data model from controllers to JSP views using the JSTL and how to design a Javascript MVC pattern, client-side, with AngularJS.

Introduction to chapter 1

Spring MVC Cookbook chapter 1, introduces a set of practices that have become Standards in the industry. From the configuration of the Eclipse IDE in order to provide an optimzed support for Java 8, Tomcat 8, Git and Maven, to a proper understanding of Maven as a build automation tool and as a dependency-management tool, you will learn how to deploy the Spring Framework on a sustainable base.

Whether a project is meant to become a massively profitable machine or to remain a rewarding experience, everyone of us, developers, should feel excited to jump into it the same way. Boosted by previous experiences, confident from the understanding of standards and reliable Enterprise-level patterns.

More than a guide through the first development stage of the book's app: CloudStreetMarket, these standard practices are presented in this chapter as a Routine for developers towards an Enterprise Spring Application.

This is the platform where can be found the latest news about the book. [PACKT] Spring MVC Cookbook will be released during December 2015. Find more details on the publisher website: Spring MVC Cookbook

CloudStreetMarket.com

CloudStreetMarket.com is a scalable Spring MVC showcase web-application developed with the book, chapter after chapter. The project is maintained and hosted on Github under a GPL license. The app is virtual trading platform with social capabilities. More details can be already be found on the Wiki

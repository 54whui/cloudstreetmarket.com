Introduction to chapter 1

Spring MVC Cookbook chapter 1, introduces a set of practices that have become Standards in the industry. From the configuration of an Eclipse IDE to get an optimized support for Java 8, Tomcat 8, Git and Maven, to a proper understanding of Maven as a build automation tool and as a dependency-management tool, you will learn how to deploy Spring Framework on a sustainable base.

Whether a project is meant to become a massively profitable machine or to remain a rewarding experience, everyone of us, developers, should feel excited to jump into it the same way: boosted by previous individual experiences, confident from Big Pictures on reliable Enterprise-level patterns.

More than a guide through the first development stage of the book's app: CloudStreetMarket, these standard practices are presented in this chapter as a Routine for developers towards Enterprise Spring Applications.
